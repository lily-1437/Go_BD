# Go BD: Your all in one travel assistance

**Go BD** is a cloud-based smart tourism management and real-time travel assistance system designed specifically for Bangladesh. The platform addresses a core problem faced by tourists today.

## One Platform, Every Service You Need
- Connects tourists with hotels, restaurants, transportation services, tour guides, and tourist spots
- Offers personalized trip planning based on traveler type:
  - Family
  - Student
  - Budget traveler
  - Solo female traveler
- Provides special safety information for vulnerable groups

## Always Up-to-Date, Always on Track
- Real-time updates on pricing, weather conditions, seasonal changes, and local events
- GPS-based location guidance to eliminate confusion caused by similar place names

## Smarter Tools for Service Providers and Admins
- Integrated dashboard for service providers to:
  - Track bookings
  - Analyze service popularity across different tourist types
- Centralized MySQL database with role-based access control for data accuracy

## Safe, Simple, and Secure Booking
- Unified booking and payment process
- No need to repeatedly enter personal and financial details across multiple platforms

## Built on a Solid Technical Foundation
- Built using XAMPP with a MySQL database
- Structured around a well-normalized relational schema
- Practical, database-driven solution for safer, smarter, and more organized travel
